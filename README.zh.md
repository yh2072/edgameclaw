<h2>为什么是游戏？</h2>

<p>不是因为学习应该<em>"有趣"</em>。</p>

<p>而是因为<strong>好的游戏设计和好的教学法，本质上是同一回事。</strong></p>

<p>一个设计精良的游戏关卡，只教你一件事——<br />
通过<strong>失败、重复、反馈和掌握</strong>——<br />
以你真正需要的顺序。</p>

<p><em>这才是人类真正的学习方式。</em></p>

## 📄 &nbsp; 上传任何材料，获得可玩课程

教科书 · 研究论文 · 内部文档 · 培训手册 · 课堂笔记 · API 文档 · 维基百科

粘贴原始材料 → ahafrog 读取并识别核心概念，将每个概念匹配到最合适的游戏机制，自动生成一系列可玩的微型课程——配有视觉效果、互动玩法和自适应节奏。

**无需游戏设计经验，无需工程技能，无需课程设计背景。**

## 如何运作

</div>

<br />

**1 →** 粘贴任何学习材料——教科书章节、内部文档、研究论文，任何文字内容。

**2 →** ahafrog 的 AI 识别核心概念，将每个概念匹配到对应的游戏机制。

**3 →** 自动生成完整可玩课程——视觉效果、互动玩法、自适应节奏——可立即发布。

**4 →** 学习者在游戏中走向掌握。每次回放都在同一心智模型上加深理解。
